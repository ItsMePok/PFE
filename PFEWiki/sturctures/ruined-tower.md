# Ruined Tower

<figure><img src="https://github.com/ItsMePok/PFE/blob/wikiAssets/structures-pixel/RuinedTower.png?raw=true" alt=""><figcaption></figcaption></figure>

## **Structure identifier:** <mark style="color:yellow;">poke\_pfe:ruined\_tower</mark>

## Generation:

* Generates in Plains biomes
* This structure can be located with the /locate command

{% hint style="warning" %}
Structures will only generate in newly generated chunks.

If you are using an older world you may need to explore further to find this structure
{% endhint %}

## Loot:

{% hint style="info" %}
A higher weight makes it more likely to be selected from that pool
{% endhint %}
