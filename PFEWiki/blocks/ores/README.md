# Ores

## General info:

* Fortune does work on the ores

## Where do Ores generate?

### **Overworld:**

* Cobalt
* Void
* Shade
* Onyx
* Holy
* Hellish
* Godly
* Galaxy
* Demonic

### **Nether:**

* Shade
* Onyx
* Hellish
* Demonic

### **End:**

* Nebula
* Void
* Godly
* Galaxy
* Astral

## But at what Y-level do ores generate?

### End & Nether: Any Y-level&#x20;

### **Overworld:**

* Cobalt: Any
* Void: -64 || -59
* Shade: -4 || 128
* Onyx: -60 || -16
* Holy: -8 || 256
* Hellish: -32 || 32
* Godly: 0 || 128
* Galaxy: 16 || 256
* Demonic: -50 || 0
