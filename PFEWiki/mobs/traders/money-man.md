# Money-Man

### Money Man

Identifier: **poke:money\_man**

#### Trades:

> \[!NOTE] The Currency is not obtainable in survival & is really only useful for economy systems in servers

**Currency Exchange:**

| Wants     | Gives   |
| --------- | ------- |
| 5x $1     | $5      |
| 2x $5     | $10     |
| 2x $10    | $20     |
| 5x $10    | $50     |
| 2x $50    | $100    |
| 5x $100   | $500    |
| $5        | 5x $1   |
| $10       | 2x $5   |
| $20       | 2x $10  |
| $50       | 5x $10  |
